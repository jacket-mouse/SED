# 社工库项目
简易社工库--可以支持Excel、CSV、TXT文件的导入，自动创建索引。

- 查询：主页面十分简洁，只有一个输入框，在其中输入想要查询的内容回车后就可以在下面输出结果中显示对应的结果。
- 文件上传：每次只能上传一个文件，文件类型包括：excel、csv、txt，上传的位置是在后端目录下相对应的文件类型文件夹下，例如：txt/、 csv/、xlsx/ 等。上传后会自动为该文件创建索引，下面是对于不同文件类型的处理。
